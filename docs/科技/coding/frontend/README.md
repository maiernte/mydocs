###### 两个class之间有空格和没有的区别

`.classA .classB {}` 如下

 ```html
<p class="classA">文字文字
    <span class="classB">pp这个class生效</span>
    ....
</p>
 ```

`.classA.classB {}` 如下

```html
<p class="classA classB">
    pp2这个class生效
</p>
```
