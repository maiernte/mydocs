**HHKB Professional BT 日语配列**

###### 特殊符号

⌘ *command*  | ⌥  *option*  |  ⌃  control  |  ⇧ *shift* 

英语输入法中输入 
- € `⌥ + ⇧ + 2`
- £ `⌥ + 3`
- 反斜杠 \ `⌥ + ¥`
- 反点 **´** ``⌥ + E` 或者 
- MD文件的代码符号(反斜点) `⌥ + ⇧ + \` 

中文输入法中

- 英语句号：在中文中，直接输入英文以及中文句号。句号后跟字母，或者点两次句号。就会自动转换成全角句号。
- 查找符号 `⇧ + ⌥ + B`   Emoji `⌘ + ⌃  + Empty`
- 间隔号 · `⌥ + \`

**休眠**：Fn + Power （*按超一秒*）



###### 原厂设置

- Fn+Q 进入配对模式，不支持多个设备。

- 更改DIP配置的时候，比如Windows或者MacOS模式，需要先给键盘断电。设置完成后打开键盘电源。

- 左边 HH 标志按键可以当 ⇪使用。

- 黄灯30闪一次表示电池量低，15秒闪一次表示需要换电池。背后开关按住两秒以上，打开/关闭键盘。
